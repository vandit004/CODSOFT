# CODSOFT Java Development Internship

This repository contains the Java projects completed during the CodSoft Java Development Internship.

## Task 1 - Number Game

A console-based number guessing game where the user attempts to guess a randomly generated number.

### Features

* Random number generation
* Attempt limit
* Score tracking
* Replay functionality

## Task 2 - Student Grade Calculator

A Java application that calculates total marks, percentage, and grade based on student marks.

### Features

* Total marks calculation
* Percentage calculation
* Grade assignment
* Input validation
* Statistics display

## Task 3 - ATM Interface

A console-based ATM simulation using Object-Oriented Programming concepts.

### Features

* Deposit money
* Withdraw money
* Check balance
* Transaction validation
* Bank account integration

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Scanner Class
* Random Class
